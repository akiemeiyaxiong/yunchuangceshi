# ALIGNMENT_部署.md - 云创网站部署对齐文档

## 1. 项目概述

### 1.1 项目信息
- **项目名称**: 云创电商网站
- **部署目标**: 腾讯云服务器 (43.166.220.37)
- **服务器配置**: Ubuntu 22.04 LTS, 2核2G, Docker 26.x 已预装

### 1.2 技术栈确认
| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | 纯 HTML/CSS/JS | 静态页面，位于 qianduan/ 目录 |
| 后端 | Node.js + Express | API 服务，位于 server/ 目录 |
| 数据库 | MySQL (外部) | 内网 IP: 10.3.4.14:3306 |
| 部署方式 | Docker + Docker Compose | 容器化部署 |
| 反向代理 | Nginx | 处理静态文件和 API 路由 |
| SSL | Let's Encrypt / 自签名 | HTTPS 配置 |

## 2. 架构决策

### 2.1 部署架构
```
用户请求
    ↓
[443/80] Nginx (反向代理 + SSL终止)
    ├── /api/* → Node.js 后端 (容器内:3000)
    └── /*      → 前端静态文件 (容器内:80)
```

### 2.2 容器设计
| 容器 | 镜像 | 端口映射 | 职责 |
|------|------|----------|------|
| nginx | nginx:alpine | 80:80, 443:443 | 反向代理、静态文件服务、SSL |
| backend | node:18-alpine | 3000:3000 | Express API 服务 |

### 2.3 数据库连接
- **类型**: 外部 MySQL (非容器化)
- **地址**: 10.3.4.14:3306
- **数据库**: yunchuang
- **连接方式**: 后端容器直接连接内网 IP
